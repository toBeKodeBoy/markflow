# MarkFlow v1.0 — 核心版

> 基于 [开发计划](./开发计划.md) | 当前发布：**v1.0.1** | 更新日期：2026-07-04

---

## 1. 版本定位

**v1.0** 是 MarkFlow 的 **首个可对外发布版本**：在 v0.0 闭环之上，交付完整的本地 Markdown 笔记体验——多视图编辑、文件夹组织、导入导出、主题与目录导航，并建立测试与 CI 基线。

**v1.0.1** 在 v1.0 基线上提前合入了部分原规划在 v1.1 / v1.2 的能力（见 §3.2），不影响 v1.0 核心范围的「已完成」判定。

---

## 2. v1.0 核心交付（✅ 已完成）

| 模块 | 实现位置 | 说明 |
|------|----------|------|
| 笔记 CRUD | `src/stores/note.ts` | 新建/重命名/删除，标题自动提取 |
| 文件夹管理 | `note.ts` + `Sidebar.vue` | 创建/重命名/删除，按文件夹过滤 |
| 四视图 | `App.vue` + Editor 组件 | 预览 / 分屏 / 源码 / 专注 |
| 内容持久化 | `useStorage.ts` | uTools dbStorage ↔ localStorage 双通道 |
| 导入/导出 .md | `preload.js` + `Toolbar.vue` | 文件对话框 / 浏览器下载 |
| 明暗主题 | `useTheme.ts` | 浅色 / 深色 / 跟随系统或 uTools |
| 目录导航 | `Toc.vue` + `useTocHeadings.ts` | 标题解析、点击跳转 |
| 格式化工具栏 | `Editor.vue` | 源码/分屏模式快捷插入 Markdown |
| 滚动同步 | `useScrollSync.ts` | 分屏左右滚动比例同步 |
| 大文件策略 | `constants.ts` + `note.ts` | >200KB 自动切分屏 |
| 复制 HTML | `Preview.vue` | 分屏预览一键复制渲染结果 |
| GFM 与语法高亮 | Milkdown + marked + highlight.js | 表格、任务列表、代码块等 |
| 自动化测试 | `tests/` | 单元 / 集成 / 架构约束（34 文件，189 用例） |
| CI | `.github/workflows/ci.yml` | `npm test` + `npm run build` |

**技术栈：** Vue 3.5 + Pinia 3 + Vite 8 + Milkdown 7 + CodeMirror 6 + marked 18 + TypeScript 6
